# Contributing

Thanks for your interest in improving R3XA_API!

## Development setup
```bash
python -m venv .venv
. .venv/bin/activate
pip install -e .[dev]
```

## Run tests
```bash
python -m pytest
```

## Build docs
```bash
pip install -r docs/requirements.txt
make docs
```

## Style
- Keep changes minimal and focused.
- Update docs/examples when you add features.
